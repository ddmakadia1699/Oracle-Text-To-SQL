  GL_LEDGERS contains information about the ledgers defined in the Accounting Setup Manager and the ledger sets defined in the Ledger Set form.  Each row includes the ledger or ledger set name, short name, description, ledger currency, calendar, period type, chart of accounts, and other information. Some columns in this table are not applicable for ledger sets.  In this case, default values would be inserted into these columns for ledger sets.
